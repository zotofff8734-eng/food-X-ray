# Идеи для стартапа: Получение грантов Microsoft Azure

Существует несколько основных способов получить кредиты Microsoft Azure (в которых доступен Azure OpenAI).

## 1. Программа для стартапов: Microsoft for Startups Founders Hub
Это самый выгодный вариант для реальных проектов. Дают от $1 000 до $150 000 кредитами.
*   **Уровень "Ideation" (Идея):** Дают $1 000. Не нужно юрлицо. Достаточно описать идею, команду и иметь профиль в LinkedIn.
*   **Уровень "Develop":** Дают $5 000. Потребуется подтверждение бизнеса (документы ИП/ООО).
*   **Как подать:** Заполнить заявку на официальном сайте Founders Hub.
*   **Бонусы:** Прямой доступ к OpenAI внутри Azure, GitHub Enterprise, Microsoft 365, кредиты на рекламу.

## 2. Бесплатный аккаунт для новых пользователей (Azure Free Account)
*   Microsoft дает $200 на первые 30 дней для тестирования любых сервисов.
*   Регистрация на azure.microsoft.com/free/.
*   **Важно:** Потребуется привязать банковскую карту для верификации. Без вашего явного разрешения (перехода на платный тариф) деньги сверх $200 списывать не будут.

## 3. Azure for Students (Для студентов)
*   Дают $100 кредитов на 12 месяцев.
*   **Главный плюс:** Не нужно привязывать банковскую карту.
*   Требуется студенческая почта (.edu или от вуза-партнера).
